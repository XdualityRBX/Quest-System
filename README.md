Roblox modular NPC quest system



FEATURES:
- Npc interaction with a cinematic-ish feel
- Typewriter dialogue system
- Collectable objects (in this example its an apple)
- Progress bar
- Quest notification
- Quest completion reward
- Tween animations


HOW IT WORKS:
1. The player interacts with the npc via proximity prompt
2. Dialogue plays with typewriter animation
3. player can accept or decline quest
4. Player collects apples
5. Player gets rewarded when apples are delivered


DEPENDENCIES:
- QuestModule
- RemoteEvent
- QuestEvent
- AppleEvent


WORKSPACE:
- NPC with ProximityPrompt
- A tree model with apples (parts) with ProximityPrompts
